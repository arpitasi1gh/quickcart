# 🛒 QuickCart - Shopping Cart App

![React](https://img.shields.io/badge/React-18-blue?logo=react)
![Vite](https://img.shields.io/badge/Vite-Fast-purple?logo=vite)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-yellow?logo=javascript)
![Status](https://img.shields.io/badge/Status-Completed-brightgreen)
![License](https://img.shields.io/badge/License-MIT-lightgrey)

A lightweight and interactive shopping cart application built with **React + Vite**, featuring dynamic cart management, routing, and persistent storage.

---

## 🚀 Live Demo

👉 https://quickcart-six-vert.vercel.app/

---

## 📌 Why This Project

- 🧠 Demonstrates **real-world e-commerce logic**
- ⚡ Built with **modern React patterns (Hooks + Context API)**
- 🔗 Implements **SPA routing with dynamic URLs**
- 💾 Includes **persistent state using localStorage**
- 📱 Designed with a **mobile-first responsive approach**

---

## ✨ Features

- 🛍️ Browse products with a clean UI
- ➕ Add to cart with quantity handling
- 🔄 Update item quantities (+ / -)
- ❌ Remove items from cart
- 🧮 Automatic total price calculation
- 🧾 Cart sidebar with smooth animations
- 🔢 Cart badge with live item count
- 🔍 Real-time search functionality
- 🗂️ Category-based filtering
- 🔗 Multi-page navigation using React Router
- 💾 Persistent cart using localStorage
- 📱 Fully responsive design

## 🧠 Key Concepts Demonstrated

- Component-based architecture
- Props vs Global State (Context API)
- useState & useEffect hooks
- Custom Hooks (useCart, useLocalStorage)
- Conditional rendering
- Event handling
- Immutable state updates
- Array methods (.map(), .filter(), .reduce(), .find())
- SPA Routing & Navigation

## 🧩 Features Breakdown

### 🧠 State Management
- Global cart state using Context API
- Cart visibility toggle
- Centralized reusable logic

### 🛒 Cart Functionality
- Smart add-to-cart (handles duplicates)
- Quantity management (+ / -)
- Remove items
- Total items & price calculation

### 🔗 Routing
- Home page (all products)
- Category page (`/category/:category`)
- Cart page (detailed summary)
- Navigation with `Link` & `useNavigate`

### 💾 Data Persistence
- localStorage integration
- Auto-load previous session cart
- JSON serialization & parsing

### 🔍 Search & Filter
- Real-time product search
- Case-insensitive filtering
- Category filtering via URL
- Empty state handling

### 🎨 UI/UX Enhancements
- Responsive across all devices
- Smooth hover effects & animations
- Slide-in cart sidebar
- Clean, modern UI design
- Empty states for better UX

## 📁 Project Setup

```bash
# Clone the repository
git clone https://github.com/your-username/quickcart.git

# Navigate to project
cd quickcart

# Install dependencies
npm install

# Run development server
npm run dev
```
